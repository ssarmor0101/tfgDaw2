# TFG

## Tematica
> Pagina de videojuegos clasicos

## Objetivos
> Es una aplicacion para poder jugar a videojuegos clasicos por la web
  y poder competir por tener la maxima puntuacion entre todos los jugadores

## Funcionalidades
- Roles
  - Admin: Podra ver todas las citas gestio
  - Cliente: Podra ver sus citas
- Vistas:
  - Home
  - Gestion de citas: lleva un historico y creacion de vista
  - Login
  - Register
  - Nosotros
  - Contacto
  - CRUD usuario y citas
  - CRUD Perfil
## Arquitectura/Tecnología
 - Frontend:
   - Figma para prototipado
   - React
   - Module CSS
   - Librerias: esto hay que completarlo..
 - Backend:
   - Laravel
   - ORM: Eloquent
   - Base de datos: MySql
 - Despliegue:
   - Docker
   - AWS Amazon
