# TFG

## Tematica
> Pagina de videojuegos clasicos

## Objetivos
> Es una aplicacion para poder jugar a videojuegos clasicos por la web
  y poder competir por tener la maxima puntuacion entre todos los jugadores

## Funcionalidades

### Roles
  - Admin: Podra acceder a todas las vistas y podra gestionar los juegos disponibles ademas de sus puntuaciones
  - Cliente: Podra acceder a los juegos para probarlos y competir

### Vistas
  - Home
  - Gestion de juegos
  - Login
  - Register
  - Nosotros
  - Contactanos
  - CRUD Admin: Juegos, usuarios, configuracion
  - CRUD Perfil

## Arquitectura/Tecnología

### Frontend
   - Figma para prototipado
   - React
   - Module CSS

#### Librerias

### Backend
   - Laravel
   - ORM: Eloquent
   - Base de datos: MySql

### Despliegue:
   - Docker
   - AWS (Amazon Web Service)
